# BehaviorBench

BehaviorBench is a human-annotated dataset for evaluating large language model (LLM) behavior across instruction-following, reasoning, and safety dimensions.

## What this dataset is
- Human-reviewed LLM responses
- Includes failure types, severity, and explanations
- Designed for post-training evaluation and regression testing

## What this dataset is not
- Not for training models
- Not a leaderboard
- Not ground-truth for factual accuracy

## Structure
- data/: Dataset files
- docs/: Taxonomy and guidelines
- examples/: Usage examples
